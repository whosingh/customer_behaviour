This project demonstrates an end-to-end data analytics workflow, from raw data ingestion to insights visualization and reporting. ## Project Steps
1. **Data Loading**
   - Imported the dataset into Python using Pandas.
   - Verified data structure and data types.

2. **Exploratory Data Analysis (EDA)**
   - Analyzed distributions, trends, and correlations.
   - Identified missing values and outliers.
   - Visualized insights using charts and graphs.

3. **Data Cleaning**
   - Handled missing and inconsistent values.
   - Removed duplicates and corrected data types.
   - Prepared clean data for analysis and SQL processing.

4. **SQL Analysis (PostgreSQL)**
   - Loaded cleaned data into PostgreSQL.
   - Wrote SQL queries to extract insights and metrics.
   - Performed aggregations, joins, and filtering.

5. **Dashboard Development**
   - Built an interactive Power BI dashboard.
   - Visualized KPIs, trends, and comparisons.
   - Enabled user-friendly filtering and exploration.

6. **Reporting & Presentation**
   - Created a structured analytical report.
   - Designed a professional presentation using Gamma.
   - Highlighted key findings and business insights.
